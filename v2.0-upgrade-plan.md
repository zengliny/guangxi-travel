## 🏗️ 技术架构升级

### 前端架构
```
v1.0 (静态)          v2.0 (动态)
┌──────────────┐     ┌──────────────────┐
│  Cloudflare  │     │  Cloudflare      │
│  Pages       │     │  Workers +       │
│  HTML/CSS/JS │     │  Pages           │
└──────────────┘     │  → Node.js API   │
                     │  → PostgreSQL    │
                     │  → Redis Cache   │
                     └──────────────────┘
```

### 推荐技术栈
| 层级 | 技术选型 | 理由 |
|------|----------|------|
| 前端框架 | React + Next.js | SSR/SSG支持、性能优秀 |
| 样式方案 | Tailwind CSS + CSS Modules | 快速开发、体积优化 |
| 状态管理 | Zustand | 轻量、简单 |
| 后端框架 | Cloudflare Workers | 免费、全球CDN、无服务器 |
| 数据库 | D1 (SQLite) | 免费、Serverless、500万行/月 |
| 认证 | Lucia + JWT | 轻量、适合Workers |
| 部署 | Cloudflare Workers + Pages | 全球CDN、低延迟、完全免费 |
| 消息服务 | 飞书机器人 | 免费、Webhook、智能客服 |

-- 用户表
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    phone VARCHAR(11) UNIQUE,
    openid VARCHAR(64),

CREATE TABLE browsing_history (

);
